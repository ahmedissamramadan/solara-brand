# SOLARA — Digital Marketing Project & Brand Strategy

Welcome to the official final submission repository for **SOLARA**, a premium-accessible handmade summer lifestyle brand. ## 🎯 Brand Strategy & Core DNA | جوهر استراتيجية سولارا

*   **UVP (Unique Value Proposition)**: The only Egyptian summer handmade brand combining premium customization with coastal lifestyle identity.
    *(البراند المصري الصيفي الوحيد الذي يجمع بين جودة الصناعة اليدوية، التخصيص بالاسم، وطابع الساحل الراقي).*
*   **Target Segment**: Women aged 18–47, Middle-to-High income class (Upper B / A class), focused in New Cairo, Sheikh Zayed, and coastal hubs (North Coast, Gouna, Dahab).
*   **Strategic Price Zone**: EGP 800–1,800 (Masstige).
*   **Emotional Positioning**: Selling *identity, memory, and emotional ownership* ("This was made for me").
